## User Stories
1. As an analyst, I want to see the sentiment label for each news article in the list view so that I can quickly identify negative news.

2. As an analyst, I want to view and respond to colleagues’ comments on a news article so that we can discuss possible responses to PR crises.

3. As an analyst, I want the system to store my previous analysis results so that I can review historical sentiment trends.

4. As an analyst, I want to adjust the sentiment score if the AI classification is inaccurate so that the analysis results can be improved.

5. As a PR manager, I want to view a dashboard showing the overall sentiment distribution of a brand so that I can evaluate the current brand health.

6. As a PR manager, I want to compare sentiment scores between our brand and competitors so that I can understand our relative reputation in the market.

7. As a PR manager, I want to bookmark important news articles so that I can easily access them later.

8. As a user, I want to search for news articles using keywords so that I can track sentiment related to specific products or topics.

9. As a user, I want to create an account and log in so that our internal sentiment analysis data remains secure.

10. As an administrator, I want to manage team member permissions so that only authorised PR staff can publish comments.

## Main Pages

### 1. Login / Signup Page

Description:
Satisfies the basic technical requirement of user authentication. Users can register and log into the system. Different roles such as employee login or administrator login can be supported.

### 2. Main Dashboard

Description:
This is the core page for brand monitoring. It contains two main sections at the top and a search function on the top right.

Overall section:
Displays key brand sentiment information.

Brand sentiment score:
Shows the overall sentiment ratio of the brand (positive, neutral, negative).

Sentiment trend chart:
A line chart placeholder that shows how brand mentions change over time.

Alert module:
Displays whether there are any recent negative news events.

Search section:
Allows users to search for other brands and view their sentiment performance.

### 3. Competitor Analysis Page

Description:
This page represents the commercial value of the system by comparing sentiment between the company and its competitors.

Key content:
Displays sentiment percentage comparisons between different brands using a bar chart.

Shows positive, neutral, and negative proportions to help users understand relative market reputation.

### 4. Media Feed Page

Description:
Displays collected news articles or comments and allows users to browse information.

Key content:
List of news titles and previews.

Each item includes a sentiment label such as Positive, Neutral, or Negative.

Users can click on a news item to view detailed information.

### 5. Analysis Detail and Team Collaboration Page

Description:
This page is opened after clicking a news item. It provides detailed sentiment analysis and supports team collaboration.

Key content:
Displays full article or summary and sentiment result.

Collaboration features:
Comment section for team discussion.
Users can reply to each other under a specific news item.
Supports tagging and marking, for example identifying misclassification or highlighting issues that require attention.
